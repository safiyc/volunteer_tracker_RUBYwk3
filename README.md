# Volunteer Tracker

#### Technologies used: Ruby, Sinatra, Capybara, PostgresSQL, HTML.

##### By Safiy Cham

## Description

This program lets user creates/updates/deletes projects and adds/updates volunteers.

## Specifications:

* User enters project.
  * Input: "Project 1"
  * Output: "Project 1"

* User enters volunteer to project.
  * Input: "Project 1: John."
  * Output: "Project 1: John."

* User deletes project.
  * Input: clicks on "delete"
  * Output: project is deleted

## Setup

* clone github
* run 'ruby app.rb' in terminal, while in project directory
* load http://localhost:4567/ on browser
